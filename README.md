Whatsapp Chat-Bot for SAP

# 🚀 SAP WhatsApp Bot (Free Architecture)

A real-time chatbot that integrates **SAP OData** with **WhatsApp Cloud API** to fetch material details and generate downloadable PDF reports 📄

---

## 🔥 Features

- 📦 Fetch material details from SAP (MATNR)
- 💬 WhatsApp chatbot interface
- 🔘 Interactive buttons (Next / Stop / Download PDF)
- 📄 Generate PDF reports instantly
- 🔐 Secure API integration using environment variables
- 💸 Designed with **100% FREE architecture (user-triggered model)**

---

## 🧠 Tech Stack

- Node.js
- Express.js
- SAP OData
- WhatsApp Cloud API (Meta)
- PDFKit
- Ngrok

---

## ⚙️ How It Works

1. User sends material number on WhatsApp  
2. Bot calls SAP OData API  
3. Returns material details  
4. User can:
   - Get another material
   - Download PDF report
